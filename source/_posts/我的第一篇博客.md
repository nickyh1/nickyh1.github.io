## 为什么写博客？

写博客是一种很好的学习方式，费曼学习法的精髓就是：**把你学到的东西教给别人**。当你能用清晰的语言解释一个概念，才说明你真正理解了它。

## 这个博客会写什么？

分为三个板块：

- **知识笔记** - 技术学习笔记，编程经验总结
- **工作经验分享** - 职场心得，踩过的坑
- **酒神精神** - 酒类品鉴，喝酒那些事

## Hexo 常用命令

```bash
# 新建文章
hexo new "文章标题"

# 本地预览
hexo server

# 生成静态文件
hexo generate

# 部署
hexo deploy
```
